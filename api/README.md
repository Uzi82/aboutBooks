
# Документация для API

## Авторизация

### Вход
__POST__ ```/api/auth/login``` - Вход в аккаунт.

Пример body запроса:
```
{
    "username": "admin",
    "password": "123"
}
```
*Возвращается JWT токен*

### Создание аккаунта
__POST__ ```/api/auth/register``` - Создание нового пользователя.

Пример body запроса:
```
{
    "username": "admin",
    "password": "123"
}
```
*Возвращается JWT токен*
